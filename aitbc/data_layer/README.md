# aitbc.data_layer

Data abstraction for AITBC applications.

## Exports

- `DataLayer` - Data access abstraction
- `MockDataGenerator` - Generate mock data
- `RealDataFetcher` - Fetch real data
- `get_data_layer` - Factory function

## Usage

```python
from aitbc.data_layer import DataLayer, get_data_layer
```
