# aitbc.state

State management for AITBC applications.

## Exports

- `StateMachine`, `ConfigurableStateMachine` - State machines
- `AsyncStateMachine` - Async state machine
- `StateTransition`, `StateTransitionError` - Transitions
- `StatePersistence`, `StatePersistenceError` - Persistence
- `StateMonitor`, `StateValidator`, `StateSnapshot`

## Usage

```python
from aitbc.state import StateMachine, StateTransition
```
