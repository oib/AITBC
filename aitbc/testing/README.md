# aitbc.testing

Testing utilities for AITBC applications.

## Exports

- `MockFactory` - Create mock objects
- `TestDataGenerator` - Generate test data
- `TestHelpers` - Common test utilities
- `MockResponse`, `MockDatabase`, `MockCache`
- `mock_async_call` - Mock async functions
- `create_mock_config`, `create_test_scenario`

## Usage

```python
from aitbc.testing import MockFactory, TestDataGenerator
```
