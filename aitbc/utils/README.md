# aitbc.utils

General utilities for AITBC applications.

## Submodules

- `aitbc.utils.json_utils` - JSON helpers
- `aitbc.utils.time_utils` - Date/time utilities
- `aitbc.utils.validation` - Input validation
- `aitbc.utils.env` - Environment variable helpers
- `aitbc.utils.paths` - Path utilities

## Usage

```python
from aitbc.utils.json_utils import load_json, save_json
from aitbc.utils.time_utils import get_utc_now
from aitbc.utils.validation import validate_address
from aitbc.utils.env import get_required_env_var
from aitbc.utils.paths import get_data_path
```
