# Explorer

The AITBC explorer allows you to browse and search the blockchain for transactions, jobs, and other activities.

## Features

### Transaction Search
- Search by transaction hash
- Filter by address
- View transaction details

### Job Tracking
- Monitor job status
- View job history
- Analyze performance

### Analytics
- Network statistics
- Volume metrics
- Activity charts

## Using the Explorer

### Web Interface
Visit [https://explorer.aitbc.io](https://explorer.aitbc.io)

### API Access
```bash
# Get transaction
curl https://api.aitbc.io/v1/transactions/{tx_hash}

# Get job details
curl https://api.aitbc.io/v1/jobs/{job_id}
```

## Advanced Features

- Real-time updates
- Custom dashboards
- Data export
